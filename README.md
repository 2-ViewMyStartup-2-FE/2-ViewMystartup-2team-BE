### 프로젝트 소개

- 기업 비교 투자 플랫폼은 투자자들에게 다양한 기업 정보를 종합적으로 제공하여, 보다 나은 투자 결정을 내릴 수 있도록 돕는 온라인 서비스입니다.
- 사용자는 개별 기업의 재무 상태, 성장 가능성, 시장 점유율 등을 쉽게 비교하고 분석할 수 있습니다.
